# Fork 실습 가이드

## 이 저장소가 뭔가요?

이 저장소는 여러분이 **첫 번째 Pull Request**를 경험해보기 위한 실습용 저장소예요.
아래 순서대로 따라하면 여러분의 이름이 이 저장소에 영원히 남게 됩니다! 🎉

---

## 실습 순서

### 1단계 — Fork
오른쪽 상단 `Fork` 버튼 클릭 → 내 계정으로 복사

### 2단계 — CONTRIBUTORS.md 수정
내 Fork 저장소에서 `CONTRIBUTORS.md` 파일 클릭 → ✏️ 연필 아이콘

아래 표에 내 행을 추가하세요:

```markdown
| 홍길동 | "오픈소스의 거인이 될 거예요!" |
```

### 3단계 — Commit
`Commit changes` → 메시지 입력 → `Commit changes`

### 4단계 — Pull Request
`Contribute` → `Open Pull Request` → `Create Pull Request`

### 5단계 — 기다리기 🙏
선생님이 merge해주면 내 이름이 공식 저장소에 올라가요!

---

## 💡 이게 바로 오픈소스 기여예요

전 세계 오픈소스 프로젝트에 기여하는 방법이 딱 이겁니다.
Fork → 수정 → PR → Merge!
